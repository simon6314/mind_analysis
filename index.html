如何取得 API 網址？
  1. 點選下方按鈕下載或複製 Google Apps Script 程式碼
  2. 建立一個 Google 試算表，點選「擴充功能」->「Apps Script」貼上程式碼
  3. 儲存並點選「部署」->「新增部署」，選取「網頁應用程式」，並設定存取權為「所有人」
  4. 部署後複製產生的 URL 並貼到上方欄位，就大功告成囉！
